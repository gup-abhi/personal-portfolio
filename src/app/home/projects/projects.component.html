Featured Project

Top 250 Movies

In this full-stack project, we developed a robust Node.js server that leverages open APIs from IMDB to retrieve and store data on the top 250 movies. Advanced data management techniques were used to optimize data storage in a SQLite database. Our Node.js APIs were designed following industry standards for reliability and performance. To enhance code maintainability and scalability, we incorporated advanced design patterns like the Singleton pattern and data object model. On the front-end, we crafted an engaging and responsive interface using Angular, along with a search functionality for easy title retrieval and detailed movie information, enhancing the overall user experience.

Featured Project

Django Blog

This Django project involved the creation of a dynamic blog website, utilizing Django framework, HTML, and Bootstrap to ensure an engaging user experience. Robust user authentication features were implemented, including user registration, login, and password reset, all facilitated through email. Users were encouraged to personalize their profiles by adding profile pictures, enhancing their engagement and identity within the platform. Content management was made seamless, allowing users to create, update, and delete posts with full control. A comprehensive commenting system was integrated, enabling users to express their thoughts on posts, with authors having moderation capabilities. The project also integrated the SummerNote editor, empowering users to customize their posts, including the addition of multimedia elements, fostering rich and interactive content creation on the platform.

  • Bootstrap
  • Python
  • SQLite
Featured Project

Space Invader Game

I took a trip down memory lane by creating my own version of the beloved classic, "Space Invader," using Python and the pygame module. In this nostalgic game, the objective is to defend our planet from an incoming alien invasion. Players are in control of a spaceship and must skillfully eliminate the alien invaders to rack up points and thwart the relentless attack. The stakes get higher as the aliens inch closer to the spaceship, and the game reaches its climax when one of them breaches the defenses. At that moment, your final score is revealed, providing a testament to your gaming prowess. It's a thrilling and engaging experience that captures the essence of arcade gaming from the past.

  • Python
  • Pygame
